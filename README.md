## 📌 Project Overview

The **Dutch Bros Coffee Sales Dashboard** is an interactive Business Intelligence solution developed in **Power BI** to analyze coffee shop sales performance. This dashboard enables users to monitor key business metrics, analyze customer purchasing behavior, identify top-performing products, compare store performance, and uncover sales trends through interactive visualizations.

The dashboard is designed to support data-driven decision-making using advanced Power BI features such as **Power Query, Data Modeling, DAX, and Interactive Reports**.

# 📈 Key Insights

- 💰 Total Sales: **$119K**
- 🛒 Total Orders: **25,335**
- 📦 Total Quantity Sold: **36,469**
- 📅 Weekday sales contributed approximately **66%** of total revenue.
- ☕ Coffee products generated the highest sales.
- 🕒 Peak customer purchases occurred during morning hours.
- 🏪 Hell's Kitchen store achieved the highest sales performance.
